#### Create a Azure Data Factory

- First Navigate to portal.azure.com 
- then search for "Data factory"
- Click on "Create a resource"
- Select "Data Factory" from the list
- Select the subscription
- Select the resource group
- Select the region
- Provide a unique name for the data factory
- Select the version
- In the next Git section you can configure now or later
